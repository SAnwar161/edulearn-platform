# GitHub Account Setup Guide

## If you don't have a GitHub account:

1. Go to https://github.com
2. Click "Sign up" in the top-right corner
3. Choose your plan (Free is fine for this project)
4. Fill in:
   - Username (choose something professional like "yourname-edulearn")
   - Email address
   - Password
5. Verify your email address
6. Complete the setup process

## If you already have a GitHub account:
Just sign in at https://github.com
